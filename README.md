## Instalação
### 1º Instale o ambiente de dev para execução local
```shell
❯ make pkg_install_from_zero_dev
```

### 2º Rodar o projeto
```shell
❯ make run
```

### 3º Rodar os testes
```shell
❯ make run_test
ou para saida em html
❯ make run_test_to_html
```
